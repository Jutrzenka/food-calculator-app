## :dragon_face: Installation project

**Clone the project:**
```
git clone https://github.com/Jutrzenka/food-calculator-app.git
```
**Go to the project directory:**
```
cd food-calculator-app
```
**Install dependencies:**
```
npm install
```
**Start the server:**
```
npm run start:dev
```
